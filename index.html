주제별 피드백
빠른 사용법

PC에서는 왼쪽 목록, 가운데 편집, 오른쪽 피드백 모아보기가 동시에 보입니다.

모바일에서는 하단 버튼으로 목록·기록·피드백·원고 화면을 전환합니다.

PC와 모바일이 같은 자료를 보려면 상단의 room 코드를 동일하게 입력하세요. 예: church-sermon

Firestore 규칙 예시 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /rooms/{roomId}/sermons/{sermonId} {
      allow read, write: if request.auth != null;
    }
  }
}
